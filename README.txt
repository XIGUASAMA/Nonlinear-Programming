1.“梯度下降法.ipynb”在Anaconda上编写，需相同软件上运行。同名.py文件可以直接在Python3 IDE上运行。
2.如果需要查看输出的中间变量建议在Anaconda上执行，如果只需得到答案建议运行.py文件
3.本程序基于无约束条件非线性规划的梯度下降法编写。
4.功能：可以求解多变量的无约束线性规划问题，输入可以为求极小或极大。
5.不足：只能求解无约束条件的非线性规划，对输入格式要求严格，尤其是输入求极大值时应该保证是多项式输入，不能存在括号。
6.测试样例见根目录同名文件，需要将其改名为“input.txt”并放置在与源代码相同的目录内，输出见“output.txt”。
7.提醒：程序输入必须同测试样例的形式，否则会出现不可预知的错误。
